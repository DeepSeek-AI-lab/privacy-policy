This extension provides AI chat functionality through DeepSeek AI. Your data handling depends on which features you use.

2. What Data We Collect

Local Storage Only (Never Transmitted)

  • Settings: Extension preferences (sidebar position, theme, AI model selection)
  • Screen dimensions: Used only to position popup windows (not stored)

URL Tracking (Optional - When "AI Sidebar Insights" is Enabled)

⚠️ Important: When you enable "AI Sidebar Insights" in options, the extension will:
  • Track URLs you visit to improve the extension
  • Send usage data to our analytics service
You can disable this anytime via the toggle in options page.

DeepSeek AI Processing

When you use AI chat features, DeepSeek AI processes:

  • Chat messages, files, and conversation history
  • Account info (if you create an account): email, username, date of birth
  • Technical data: IP address, browser type, device info

See DeepSeek's Privacy Policy for details.

Aitopia.ai (Technology Partner)

May process extension usage patterns, technical data, and user content. See Aitopia's Privacy Policy.

3. How We Use Data

  • Extension functionality: Provide AI sidebar, remember preferences, position windows
  • Analytics (when enabled): Understand which websites users find the extension useful on
  • Service delivery: Connect you to DeepSeek AI and backend services

4. Data Sharing

External Services We Connect To:

  • DeepSeek AI - AI chat service
  • Aitopia.ai - Extension backend services
  • Analytics service - Usage analytics (only when "AI Sidebar Insights" is ON)

We do NOT:

  • Track browsing history (unless you enable "AI Sidebar Insights")
  • Read page content (only URLs when tracking is enabled)
  • Sell your data to third parties

5. Your Privacy Controls

Stop URL Tracking:

Turn OFF "AI Sidebar Insights" toggle in extension options page

Stop DeepSeek Data Collection:

Don't use the AI chat features, or manage your data through DeepSeek's platform

Delete All Extension Data:

Uninstall the extension or clear data through Chrome settings

Privacy Tip: If concerned about browsing history, verify "AI Sidebar Insights" is OFF in options.

6. Data Retention

  • Local data: Stays on your device until you uninstall or clear extension data
  • DeepSeek data: Retained per their privacy policy
  • Analytics data: Retention managed by analytics provider

7. Extension Permissions Explained

  • Storage: Save your preferences locally
  • System Display: Get screen size to center popup windows
  • Tabs: Monitor tab activity when "AI Sidebar Insights" is enabled
  • Content Scripts (All URLs): Inject AI sidebar when you activate it
